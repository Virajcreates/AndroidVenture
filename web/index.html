About This Demo

This web viewer demonstrates real-time edge detection using your computer's webcam. The same edge detection algorithm (Sobel operator) is used as in the Android app, but implemented in JavaScript instead of OpenCV C++.

Tech Stack:

  • HTML5 Canvas for frame rendering
  • WebRTC getUserMedia API for webcam access
  • Sobel edge detection algorithm in JavaScript
  • Real-time processing with requestAnimationFrame
